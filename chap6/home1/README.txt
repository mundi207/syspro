1.
- rw-rw-r-- 1 mundi mundi 0 Nov 11 20:45 test.txt

- : 파일 유형을 뜻한다. 만약 해당파일이 디렉터리라면 d가 들어간다.
rw-rw-r-- : 파일의 접근권한을 뜻한다. 해당파일은 소유자와 그룹은 읽기 쓰기 권한을 가지고 있고
			다른 사용자는 읽기 권한만 가진다.
1 : 링크의 수이다.
mundi : 해당 파일의 소유자 이름이다.
mundi : 해당 파일을 소유한 그룹 이름이다.
0 : 파일 크기이다. 해당 파일은 빈 파일이므로 0이다.
Nov 11 20:45 : 파일의 최종 수정한 일시이다.
test.txt : 파일 이름이다.
